# Water Meters Dashboard

SPA для просмотра и управления счётчиками (тестовое задание). Интерфейс на русском языке.

## Что реализовано

- Таблица счётчиков с колонками: тип (с иконкой), дата установки, автоматический, показания, адрес, примечание
- Пагинация по 20 записей на страницу
- Удаление счётчика с блокировкой кнопок на время запроса
- После удаления строка убирается из таблицы; при необходимости подгружается следующая запись без перезагрузки всей страницы
- Загрузка адресов отдельным запросом с кэшированием
- Состояния загрузки, ошибки и пустого списка

## Стек

React 19 · TypeScript · Vite · MobX State Tree · Axios · SCSS Modules

Архитектура — [Feature-Sliced Design](https://feature-sliced.design/) (`app`, `pages`, `widgets`, `features`, `entities`, `shared`).

## API

Данные берутся из тестового API EIS24. В режиме разработки запросы проксируются через Vite (`/api` → `showroom.eis24.me`).

| Метод  | Эндпоинт                         | Назначение              |
| ------ | -------------------------------- | ----------------------- |
| GET    | `/api/meters/?limit=&offset=`    | Список счётчиков        |
| GET    | `/api/areas/?id__in=`            | Адреса помещений        |
| DELETE | `/api/meters/{id}/`              | Удаление счётчика       |

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

```bash
npm run build   # production-сборка
npm run preview # просмотр сборки
```

## Структура

```
src/
├── app/              # точка входа, провайдеры, глобальные стили
├── pages/            # страницы
├── widgets/          # таблица счётчиков
├── features/         # бизнес-логика списка (store)
├── entities/         # модель и UI сущности «счётчик»
└── shared/           # API, утилиты, UI-компоненты
```
